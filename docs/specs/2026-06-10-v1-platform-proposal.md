# 低空 — AI 驱动无人机高速智能巡检平台 v1.0

> 以 AI 视觉感知为大脑、无人机为空中执行终端、固定摄像头为地面锚点的城市立交实时风险监测与智能响应平台。

---

## 一、产品定位

**一句话：** 用 AI 替代人工盯屏，7×24 自动检测、分级、推送、调度，让 1 个值班员管 9 个立交。

### 当前规模

| 指标 | 数值 |
|------|:--:|
| 监控点位 | 9 个重庆主城立交 |
| 摄像头 | 9 路（每立交 1 路） |
| 无人机机舱 | 4 个（覆盖半径 5km） |
| AI 检测周期 | 300s/批 × 3 事件 |
| 事件类型 | 5 类（事故/拥堵/障碍物/烟雾/火焰） |

### AI 模块明细

| 模块 | 算法/技术 | 输入 | 输出 |
|------|------|------|------|
| **检测模拟器** | 加权随机 + 场景匹配 | 12 场景素材库 | 事件 + SVG 标注截图 |
| **异常检测** | Z-score（7天逐时） | 168 个时段事件数 | 异常时段 + 严重度 |
| **空间聚类** | 简化 DBSCAN（Haversine 2.5km） | 事件坐标 | 热点区域 + 立交名 |
| **置信度校准** | 自适应 Platt Scaling | 原始置信度 | 校准后置信度 |
| **趋势预测** | Holt-Winters（trend+seasonal） | 168h 时序数据 | 24h 预测 + 趋势方向 |
| **风险评分** | 多维加权（6因子） | 实时事件状态 | 0-100 风险指数 |
| **对话助手** | DeepSeek V3 / Mock 降级 | 用户文本 | Markdown 回复 |
| **截图生成** | SVG 绘制（场景+标注框） | 场景元数据 | 640×360 标注截图 |
